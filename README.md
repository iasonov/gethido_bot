# gethido_bot

## Что это
Telegram-бот для запуска рассылок по группам/чатам учебных программ. Бот принимает одно сообщение от администратора, а затем отправляет его в выбранные чаты через Telegram Bot API.

## Для кого
Для администраторов и менеджеров, которым нужно быстро разослать одно и то же сообщение нескольким программам с фильтрацией по параметрам.

## Что умеет
- Принимает сообщение для рассылки от пользователя из списка `ADMIN_IDS`.
- Пошагово собирает фильтры: уровень программы, партнёрский статус, кампус и участие в раннем приглашении.
- Даёт вручную включать и выключать отдельные программы перед отправкой.
- Отправляет сообщение в целевые чаты через `copyMessage`.
- Повторяет отправку при `Timeout` и делает несколько попыток.
- Сохраняет состояние пользователя и промежуточные данные в JSON.
- Пишет сводку о рассылке для администраторов и сохраняет лог.

## Как это устроено
- `gethido_bot.py` — основной скрипт и вся логика бота.
- `programs_real.csv` — источник данных по программам, включая `tg_chat_id`, `level`, `partner`, `campus`, `early_invitation`.
- `user_states.json` / `user_states.json.bak` — хранение состояния и выбранных фильтров между перезапусками.
- `logs.txt` — журнал рассылок.
- `my_secrets.py` — хранение `BOT_TOKEN`.
- Telegram Bot API — внешний сервис для чтения обновлений, отправки сообщений и копирования сообщений в чаты.

Поток данных:
1. Админ пишет боту `/start` или сразу отправляет сообщение.
2. Бот сохраняет текст и `message_id`, затем просит выбрать фильтры.
3. Фильтры применяются к `programs_real.csv` через `pandas`.
4. По inline-кнопкам формируется список целевых программ.
5. После подтверждения бот копирует исходное сообщение в каждый `tg_chat_id`.
6. Итог отправляется администраторам и записывается в лог.

## Как запустить
1. Укажите `BOT_TOKEN` в `my_secrets.py`.
2. Установите зависимости. Не найдено в репозитории: `requirements.txt`, поэтому по импортам нужны как минимум `pandas` и `requests`.
3. Запустите бота:

```bash
python gethido_bot.py
```

4. Проверка: `python -m unittest test_gethido_bot.py`.
